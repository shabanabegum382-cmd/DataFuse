## Features

| Tool | Description |
|------|-----------|
| **File Concatenator** | Merge multiple **PDF, XLSX, CSV** files into one Excel |
| **Monthly PJP Generator** | Auto-generate **monthly route plans** |
| **Floater Incentive Tracker** | Smart **floater scheduling** with weekly off |
| **Catalogue XLookup** | Match **SOH data** with catalogue using `Description` |

---

## 1. File Concatenator

> **Merge multiple files into one Excel**

**Important:**  
All files must have **same column headers**

---

## 2. Monthly PJP Generator

> **Generate monthly route plan from store data**

**Input Format Example:**

| Plan     | Store 1                          | Store 2                        | Store 3                     | Store 4                             |
|----------|----------------------------------|--------------------------------|-----------------------------|-------------------------------------|
| Route-1  | HEALTH & GLOW - TOWLICHOWKI, HYD | HEALTH & GLOW - ALKAPURI, HYD  | CENTRO - KUKATPALLY, HYD    | HEALTH & GLOW - SUJANA FORUM MALL, HYD |

> **Column names must match exactly**

---

## 3. Floater Incentive Tracker

> **Auto-schedule floaters with weekly off rotation**

**Input Format Example:**

| Counter_Code | Store_Name                     |
|--------------|--------------------------------|
| CTR001       | HEALTH & GLOW - TOWLICHOWKI    |
| CTR002       | HEALTH & GLOW - ALKAPURI       |

> **Only `Counter_Code` & `Store_Name` required**

---

## 4. Catalogue XLookup

> **Match SOH data with catalogue using `Description`**

- Upload **Catalogue** + **SOH** files  
- Matching via **Description** column  
- **Unmatched records included** in output
